このテーマには [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) の統合がインストールされ、`astro.config.mjs` で設定済みです。MDX を使わない場合は、設定からこの統合を削除して無効化できます。

## なぜ MDX？

MDX は JavaScript と JSX を埋め込める Markdown の拡張です。Markdown コンテンツに UI コンポーネントを混在させ、インタラクティブなグラフやアラートを実現できます。

既存の MDX コンテンツがある場合、この統合により Astro への移行がスムーズになります。

## 例

以下は MDX 内で UI コンポーネントをインポートして使う例です。  
ブラウザで開くと、下のクリック可能なボタンが表示されます。

import HeaderLink from '../../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	Embedded component in MDX
</HeaderLink>
