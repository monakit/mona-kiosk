# MonaKiosk

A modern Astro integration for monetizing your content with [Polar.sh](https://polar.sh).

You can find a live showcase here: [Mona](https://www.mymona.xyz/)

## What is MonaKiosk?

MonaKiosk makes it easy to add paywalls and monetization to your Astro-powered content. Whether you're selling blog posts, courses, templates, or subscriptions, MonaKiosk handles the heavy lifting so you can focus on creating great content.

## Key Features

- **Content Paywalls** - Protect your premium content with automatic paywall generation
- **Flexible Pricing** - Support for both one-time purchases and subscriptions
- **Simple Setup** - Add a few lines to your Astro config and you're ready to go
- **Automatic Sync** - Your content automatically syncs to Polar at build time
- **Built-in Auth** - Email-based authentication and session management included

## Repository Structure

This is a monorepo containing:

- **packages/mona-kiosk** - The main Astro integration package
- **packages/demo** - Example implementation showing MonaKiosk in action

## Quick Start

```bash
npm install mona-kiosk
```

Add to your Astro config and start monetizing your content. Check out the [full documentation](./packages/mona-kiosk/README.md) for detailed setup instructions.

## Use Cases

- Selling premium blog posts and articles
- Subscription-based content access
- Digital product downloads
- Course and tutorial monetization
- Membership and sponsorship tiers
